## 실시간 구글 로그인 방명록

### 📘 파이어베이스 방명록 구축 매뉴얼 개요

* **준비물:** 구글 계정, 깃허브 로그인, 파이어베이스 로그인,  Vercel 로그인, 브라우저가 준비된 컴퓨터
* **핵심 요약:**
1. **1단계 (서버 만들기):** 파이어베이스에서 클릭 몇 번으로 주소(Config)를 따고 구글 로그인을 켭니다.
2. **2단계 (내 주소 연결):** 제공된 마법의 HTML 소스코드에 내 주소(Config)만 복사해 넣습니다.
3. **3단계 (인터넷에 띄우기):** 깃허브에 올리고 Vercel 단추를 눌러 라이브 주소(`아이디.vercel.app`)를 발급받습니다.
4. **4단계 (보안 설정):** 내 주소에서만 로그인이 되도록 도메인을 승인합니다.
------
초보자도 곧바로 따라 할 수 있도록 가장 간결하고 명확하게 정리한 매뉴얼과 AI 프롬프트입니다. 이 내용을 그대로 전달해 주시면 됩니다.

### 1. 웹 서비스 가입 및 로그인 방법

모든 서비스는 **동일한 구글 계정**으로 연동하여 로그인하면 매우 편리합니다.

* **구글 계정:** 인터넷 브라우저(크롬 등) 우측 상단 프로필을 눌러 로그인해 둡니다.
* **깃허브 (GitHub):** [github.com](https://github.com/) 접속 $\rightarrow$ **[Sign up]** 클릭 $\rightarrow$ 구글 이메일로 가입하거나 가입 후 구글 계정을 연동하여 로그인합니다.
* **파이어베이스 (Firebase):** [console.firebase.google.com](https://console.firebase.google.com/) 접속 $\rightarrow$ 이미 로그인된 구글 계정으로 자동 로그인됩니다.
* **Vercel (버셀):** [vercel.com](https://vercel.com/) 접속 $\rightarrow$ **[Sign Up]** 클릭 $\rightarrow$ 반드시 [Continue with GitHub]를 선택해 로그인합니다.

### 2. 1~3단계 구현용 AI 프롬프트 (구글 로그인 포함)

아래 박스 안의 내용을 그대로 복사해서 AI(Cursor, ChatGPT 등)에게 던지면 올바른 코드를 즉시 짜줍니다.

```text
HTML, CSS, 바닐라 자바스크립트(ES6) 딱 한 파일(index.html)로 구성된 "실시간 구글 로그인 한 줄 방명록" 앱 소스코드를 짜줘.

[필수 요구사항]
1. Firebase v10+ Web SDK(CDN 방식)의 Auth 및 Firestore 연동할 것.
2. 상단에 [구글 계정으로 로그인하기] 버튼을 만들고, 로그인 성공 시 사용자의 구글 프로필 실명과 [로그아웃] 버튼, 그리고 글 입력창이 보이게 해줘. (로그인 전에는 입력창 숨김)
3. 사용자가 글을 쓰고 등록을 누르면 Firestore의 'guestbook' 컬렉션에 구글 실명(author), 메시지(message), 작성시간(createdAt)이 저장되어야 해.
4. onSnapshot 함수를 사용해서 새로고침 없이도 등록된 방명록 리스트(작성자 실명과 메시지)가 화면에 실시간으로 정렬되어 나타나게 해줘.
5. 디자인은 아주 심플하고 모던한 화이트&그레이 톤으로 만들어줘.
6. `const firebaseConfig = { ... }` 부분은 내가 내 키를 직접 채워 넣을 수 있게 빈칸으로 완전히 비워두고 짜줘.

```

이 프롬프트로 얻은 코드에 파이어베이스 키를 넣고 깃허브를 통해 Vercel에 올리면 완성됩니다.
